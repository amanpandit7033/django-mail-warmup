# Django Mail Warmup System

This project is a Django-based mail warmup engine.

## Features

- Multiple sender emails with app passwords
- Recipient, Subject, Body management
- HTML email support
- Time gap control
- Daily limit per sender
- Start/Stop warmup from UI
- Warmup logs
- CSV bulk upload
- Background warmup runner

## Run

```bash
pip install -r requirements.txt
python manage.py dev
